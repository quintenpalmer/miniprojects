Algorithms
==========

linked_list.py
---------------
Demonstrates the following:

 - reversing linked lists
 - finding cycles in linked lists

parse.py
--------
Useful for formatting flattened data structures that contain nested parenthesis

e.g.

    struct(4,true,nested_struct(true,"hi"))

will be output as:

    struct(  
        4,  
        true,  
        nested_struct(  
            true,  
            hi  
        )  
    )
